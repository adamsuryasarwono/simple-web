Tugas 12 - Platform as a Service (PaaS)

Deskripsi Tugas

Website ini dibuat untuk memenuhi Tugas 12 mata kuliah Cloud Computing, yaitu mencari penyedia layanan PaaS gratis dan membuat aplikasi sederhana menggunakan layanan tersebut. Aplikasi ini dideploy menggunakan Vercel sebagai platform PaaS.

Tentang Vercel

Vercel adalah platform PaaS yang digunakan untuk melakukan deployment aplikasi web secara cepat dan mudah. Vercel mendukung aplikasi statis seperti HTML dan CSS serta framework modern. Pengguna tidak perlu melakukan konfigurasi server secara manual.
